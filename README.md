# جهانگیری — فروشگاه قطعات یدکی خودرو

فروشگاه آنلاین قطعات یدکی خودرو با Flutter Web و FastAPI.

## اجرا

### Backend

```powershell
cd backend
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend

```powershell
cd frontend
flutter pub get
flutter run -d chrome
```

## امکانات

- فروشگاه با فیلتر پیشرفته، مرتب‌سازی و صفحه‌بندی (۲۰ محصول در هر صفحه)
- مگا منوی دسته‌بندی
- اسلایدر صفحه اصلی، دسته‌بندی خودرو و قطعات
- ثبت‌نام / ورود / فراموشی رمز
- سبد خرید، تسویه حساب، علاقه‌مندی‌ها
- پنل کاربری: سفارش‌ها، پروفایل، آدرس‌ها، تیکت، اعلان‌ها
- پنل مدیر: داشبورد، سفارش‌ها، محصولات، گزارش فروش، تیکت‌ها
- صفحات درباره ما و تماس با ما
